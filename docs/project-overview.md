# Project Overview

## Problem Statement
Freshers often struggle to track multiple job applications, coding progress, interview rounds, and learning resources across different platforms such as spreadsheets, notes, and emails.

This scattered workflow leads to missed deadlines, poor preparation visibility, and inefficient interview follow-up.

The Placement Preparation & Interview Tracker System solves this by centralizing all placement activities into one structured dashboard.

## Objective
The objective of this project is to help students and fresh graduates manage:
- job applications
- DSA preparation
- interview schedules
- learning resources
- resume and GitHub links

in one place

## Modules
1. User Authentication
2. Job Application Tracker
3. DSA Progress Tracker
4. Interview Scheduler
5. Resource Manager
